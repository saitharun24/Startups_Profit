A Basic Machine Learning Model using Multivariate Linear and Multivariate Polynomial Regression¶

Here we will be looking at the various basic data cleaning techniques, visualization techiniques and then finally build a model, to predict the profits of new startup companies, using Multivariate Linear Regression(a type of linear regression in which more than one independent variable (predictors) and more than one dependent variable (responses), are linearly related)and Multivariate Polynomial Regression(a type of polynomial regression in which more than one independent variable (predictors) and more than one dependent variable (responses), are polynomially(eg: quadratic,cubic,etc.,.) related).

We will then go ahead to compare between Linear and Polynomial regression, to see which model gives a better result for our dataset.

Multivariate regression is a technique that estimates a single regression model with more than one outcome variable. When there is more than one predictor variable in a multivariate regression model, the model is a multivariate multiple regression.

Predicting the Profit using Multivariate Linear Regression¶

Multivariate Linear Regression is quite similar to the simple linear regression model, but with multiple independent variables contributing to the dependent variable and hence multiple coefficients to determine and complex computation due to the added variables.

Multivariate Linear Regression in a quadratic equation is of the form

y = m1.a1 + m2.a2 + m3.a3 + ..... + mn.an + a0 where '.' represents multiplication

Mean squared error regression loss is a a non-negative floating point value(0.0 being the value of an ideal model), which is a measure the average of the squares of the errors—that is, the average squared difference between the estimated values and the actual value.

Mean squared logarithmic error regression loss is a non-negative floating point value(0.0 being the value of an ideal model), which is a measure of the relative difference between the real and predicted values.

R2(coefficient of determination) regression score function is a floating-point value which can also be negative(1.0 being the value of an ideal model), which is a statistical measure of how close the data are to the fitted regression line. It is also known as the coefficient of determination, or the coefficient of multiple determination for multiple regression.

Mean absolute percentage error regression loss (MAPE), is non-negative floating point(0.0 being the value of an ideal model), which is a measure of how accurate a forecast system is. It measures this accuracy as a percentage, and can be calculated as the average absolute percent error for each period minus actual values divided by actual values.

Predicting the Profit using Multivariate polynomial Regression¶

In Polynomial Regression, the relationship between the response and the Regressor variable is modelled as the nth order polynomial equation. It is used when the response variable is non-linear, i.e., the scatter plot gives a non-linear or curvilinear structure.

Multivariate Polynomial Regression in a quadratic equation is of the form

y = B0 + B1.x1 + B2.x2 + B3.x3 + ..... + Bn.xn + c where xi = x^i and '.' represents multiplication

Some conclusions we can make are¶

  1. From the accuracy measure statistics we can see that as the degree of polynomial, being used to build the model, increases our model's accuracy tends to decrease. This can be seen from the r2 score of polynomial degree 1 (linear polynomial model) which is 0.867, r2 score of polynomial degree 2 (quadratic polynomial model) which is 0.748 and r2 score of polynomial degree 3 (cubic polynomial model) which is -388.979(hell of an error i.e the model is completely wrong). Based on the r2 and other metrics scores we can confidently say that as the degree increases model becomes worse off (this can be seen from degree 3 onwards).

  2. Since polynomial of degree 1 is a linear one, which is same as the one we obtained using multivariate linear regression, we can say that for out dataset multivariate linear regression is the best model. This is because we are only dealing with 4 variables (3 independant and 1 dependant) which correlate with each other directly and in a simple way (i.e if any one increases the dependant variable also increases), and it is scenarios like this that are what are best predicted using multivariate linear regression.

  3. Polynomial regression works best when there are many variables and the relationship between them are kind of complex, more the complexity of the relationship amongst variables the better a model built using a higher degree polynomial would perform.

Thus we have compared between Multivariate Linear regression and Multivariate Polynomial regression, thereby proving that for our dataset (a simple dataset on profits made by startups) the best model would be that obtained through Multivariate Linear regression.¶
